How to use this page

This starter is built for QR-code traffic: quick to load, easy to scan, and simple to update. The layout mirrors the postcard look with a strong branded header, a rounded hero card, and a clear call to action near the top.

  • Make the exchange model obvious: learners support each other.
  • Use the same warm visual family as Friends without Frontiers while giving this page its own title and CTA path.
  • A short explainer section can describe how matching works and what participants should expect.
  • Keep content modular so the team can adapt this into a sign-up flow later.

Shared but separate

This file intentionally repeats layout markup so it stays independent, even though it shares most styling with the Friends page.

Future enhancement

You could later add a lightweight enquiry form or an external matching link without changing the page shell.
